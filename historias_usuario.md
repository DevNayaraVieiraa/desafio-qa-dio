# Histórias de Usuário - SwagLabs

## História 1: Login na Loja
**Como** cliente da loja SwagLabs
**Quero** realizar login com meu usuário e senha
**Para** que eu possa acessar a lista de produtos e fazer compras

**Critérios de Aceite:**
- O sistema deve validar usuário e senha cadastrados.
- Se o login for válido, redirecionar para a página de produtos.
- Se o login for inválido, exibir mensagem de erro "Username and password do not match".

## História 2: Adicionar Produto ao Carrinho
**Como** cliente logado
**Quero** adicionar um produto ao meu carrinho
**Para** que eu possa comprá-lo posteriormente

**Critérios de Aceite:**
- O botão "Add to cart" deve mudar para "Remove" após o clique.
- O ícone do carrinho deve exibir o número de itens adicionados.
